<center><h1>Tin Whistle Helper - Manual</h1></center>
<center><h3><i>alpha version</i></h3></center>

Tin Whistle Helper helps you practice playing your tin whistle in D 
and lets you make and edit the tabs. Below are the controls to use this tool.

<H2><u>Toolbar</u></H2>
<code>
[<]    : Show file list to load a file (file list stays open) 
[Load ]: Load (filedialog) a tabs file (.tb) or abc notated file (.abc)
[New  ]: Start with clean file and resets beats per minute to 120
[Save ]: Save a tabs file under the name you provide
[SShot]: Save a screenshot of all tabs.
[<]    : Half number of beats per minute (bpm)
[◀]    : Reduce number of beats per minute by 5
[▶]    : Reduce number of beats per minute by 5
[>]    : Double number of beats per minute 
 ⊠ CO  : Count Off 4 beats after pressing play before starting song
 ⏅     : Sets number of beats for metronome to sound
 ⊠ ♫⚟  : Play the notes of the song
 ⊠ ⮤↴  : Play decorations (experimental)
 ⊠ Low : Play the song one octave lower to resemble a low whistle
[▶]    : Play song from current position of beat cursor
[■]    : If playing, stops song and set beat cursor to starting beat
         If not playing, resets beat cursor to first beat
[▮▮]   : Pause song / Resume song 
[@]    : Unroll repeated parts (needed for linear play)
[⬏]    : Switch from page view (wrapped) to linear view
[⤢]    : Autosize window and zoom level
[↙]    : Reduce zoom level to fit window size
[↗]    : Enlarge window to fit page size
ⴱ / ⴲ : Change zoom level (tab size) of page
(?)    : Show this help
</code>
<hr>
</hr>

<H2><u>Tab Area</u></H2>

<b>Mouse - Change View </b>
<code>
Left Button     : Set beat cursor to mouse pointer and plays note
Drag / Wheel    : Scroll page
[Ctrl]-Wheel    : Zoom page
Right Button    : Reset page view
</code>

<b>Keyboard - Edit Song</b>
<code>
<u><i>Navigate/play</i></u>
 Arrow keys      : Move beat cursor 
 [Tab]           : Play/Resume
 q               : Stop song 
 w               : Pause/Resume 
 r               : Autosize/-zoom 

<u><i>Notes</i></u>
 d,e,f,g,a,b     : 1st oct d..f#..b
 c,D,E,F,G,A,B   : 2nd oct c#..f#..b
 C               : 3td-oct c#
 [Alt]-d,f,g,a,c : 1st-oct d#,f,g#,a#,c
 [Alt]-D,F,G,A,C : 2nd-oct d#,f,g#,a#,c
 1...9           : length of note
 _               : change note to rest
 [Ins]           : Insert rest before
 [Shift][Ins]    : Insert rest after
 [Alt]-3         : set length 2/3 beat
 [Scroll Lock]   : autoadvance note

<u><i>Decorators</i></u>
 > , < , ^ , =   : tap/strike ↴, cut ⮤,
                   roll ⮤↴, slide ⇒ 
 @ , ~ , * , :   : tongue ᳅, vibrato ∿, 
                   staccato *, legato -
 [Escape]        : Remove decorator

<u><i>Repeats</i></u>
 { , }           : start/end repeat
                   [Ctrl]/[Alt] 2x,3x.
                   [Shft] after cursor  
 ?               : Note repeat 1 less 

<u><i>Delete</i></u>
 [Del]           : Remove note/repeat 
 [Shift][Del]    : Remove after cursor
 [Ctrl]-z        : Undo last delete 

<u><i>Visuals</i></u>
 [F1]..[F10]     : Tab color
 [Shift][Fx]     : Page color
 [Space],|       : Visual space/v.line
 [Backspace]     : Remove space/v.line
 [Return]        : Break line before 
 [Shift][Return] : Break line after 
 [Backspace]     : Join if at 1st tab

<u><i>Export</i></u>
 p               : Screenshot vis. tabs  
 [shift]-p       : Screenshot ALL tabs 
</code>

<u>Remarks</u>:
- When you change a note and the last tab is not a rest, a rest 
  will be appended to ensure you keep enough empty note space 
  to enter new notes.
- Title will be changed when saving the file (to filename)
- Repeat start and end should be on same row of tabs
</code>
